## Kubernetes Security – Wichtige Referenzen

| Ressource | Beschreibung |
|-----------|--------------|
| [OWASP Kubernetes Security Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Kubernetes_Security_Cheat_Sheet.html) | OWASP Best Practices zur Absicherung von Kubernetes-Clustern |
| [Kubernetes Security Checklist](https://kubernetes.io/docs/concepts/security/security-checklist/) | Offizielle Kubernetes-Checkliste mit grundlegenden Security-Empfehlungen |
| [NSA / CISA Kubernetes Hardening Guide](https://media.defense.gov/2022/Aug/29/2003066362/-1/-1/0/CTR_KUBERNETES_HARDENING_GUIDANCE_1.2_20220829.PDF) | Hardening-Empfehlungen für Kubernetes-Cluster aus Sicht staatlicher Sicherheitsbehörden |
| [CIS Kubernetes Benchmark](https://www.cisecurity.org/benchmark/kubernetes) | Industriestandard für sichere Konfiguration von Kubernetes-Komponenten |
| [MITRE ATT&CK for Containers](https://attack.mitre.org/matrices/enterprise/containers/) | Angriffstechniken und Bedrohungsmodell für Container-Umgebungen |
| [Falco Runtime Security](https://falco.org/) | Runtime-Security-Engine zur Erkennung verdächtiger Container-Aktivität |
| [Trivy Security Scanner](https://github.com/aquasecurity/trivy) | Scanner für Container-Images, Filesysteme und Kubernetes |
| [Kyverno Policy Engine](https://kyverno.io/) | Kubernetes Policy Engine zur Durchsetzung von Security-Policies |
| [Open Policy Agent (OPA)](https://www.openpolicyagent.org/) | Policy-Framework zur Validierung und Kontrolle von Deployments |
| [Sigstore / Cosign](https://sigstore.dev/) | Signierung und Verifikation von Container-Images (Supply-Chain-Security) |
| [kube-bench](https://github.com/aquasecurity/kube-bench) | Tool zur Prüfung eines Clusters gegen den CIS Kubernetes Benchmark |
| [kube-hunter](https://github.com/aquasecurity/kube-hunter) | Tool zur Simulation von Angreifern und zur Erkennung von Schwachstellen |
